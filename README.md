# Hostel Booking System (.NET Backend)

## 🚀 Features
- User Authentication (JWT)
- Hostel Listing
- Room Booking
- Feedback & Ratings

## 🛠 Tech Stack
- .NET Web API
- SQL Server
- React (Frontend)

## ⚙️ How to Run
1. Open project in Visual Studio
2. Run the API
3. Test using Swagger

## 📌 Future Improvements
- Payment Integration
- Map-based Search
- Notifications
